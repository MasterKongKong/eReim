举例：某一个P & L Unit下有三个部门，六个人，某一费用项目的预算分配如下
Unit
$5,000
Department-1 $4,000
User A $3,000
User B 未分配个别预算
User C 未分配个别预算
Department-2 未分配部门预算 User D 未分配个别预算
User E 未分配个别预算
Department-3 未分配部门预算 User F 未分配个别预算
User G 未分配个别预算
Case 1: 当User A申请该费用时，系统会先以个人预算 $3,000判断是否Over Budget，虽然未超过预算，则会再以Departmant-1的预算 $4,000判断，虽然还是未超过部门预算，则会以全站预算  $5,000进行判断
Case 2: 当User C申请该费用时，系统会先以Departmant A的预算 $4,000判断，虽然未超过部门预算，则会以全站预算  $5,000进行判断
Case 3: 当User G申请该费用时，因未编列个人及部门预算，系统会以 $5,000 判断是否Over Budget
Case 4: 假设本站的各使用者该费用已申请金额如下列
